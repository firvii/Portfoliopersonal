The Manager

Aplicación de gestión y control financiero orientada al usuario final.

El Problema Operativo: Usuarios sin experiencia financiera enfrentaban procesos manuales y confusos al intentar entender su situación económica real.
Stack Tecnológico: Flask, Python, SQLAlchemy, JavaScript, Bootstrap, PostgreSQL.

Solución Implementada: Plataforma web parcial construida con Flask que centraliza datos financieros y ofrece métricas accionables. El backend está diseñado para integrar múltiples fuentes de transacciones, normalizar resultados y exponer paneles de control sencillos. La arquitectura prioriza la estabilidad del procesamiento de datos y la capacidad de escalar información sin bloquear la experiencia del usuario.

Resultado: Se redujo la complejidad operativa para el usuario al eliminar tareas manuales de conciliación y visualización. El enfoque permitió transformar datos dispersos en un flujo continuo de decisiones financieras, con un sistema inicialmente estable y fácil de ampliar.

Prometheus

Procesador de datos médicos para navegación rápida y consulta diaria.

El Problema Operativo: La información médica estaba fragmentada y lenta de acceder, lo que obligaba a los usuarios a perder tiempo navegando entre fuentes y formularios.
Stack Tecnológico: Python, Flask, REST APIs, pandas, HTML/CSS, JavaScript.

Solución Implementada: Procesador de datos que unifica registros médicos en un panel de navegación ágil, con filtros rápidos y respuestas inmediatas. La solución se construyó con una arquitectura centrada en consultas y visualización, para que el usuario pueda acceder a datos críticos sin pasos innecesarios.

Resultado: Se consiguió una reducción del tiempo de búsqueda de información médica, mejorando la experiencia diaria. El proyecto está diseñado como un acompañante diario, con foco en la estabilidad y la entrega de datos relevantes de forma consistentemente rápida.
